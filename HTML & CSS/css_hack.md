## CSS 헥이란

- 모든 CSS 코드는 브라우저의 구현 차이나 버그로 인해 웹브라우저마다 조금씩 다르게 보일 수 있다.

- 이 점을 이용하여 특정 브라우저를 제외하거나, 특정 브라우저에서만 CSS를 적용하는 기법을 <strong>CSS 핵</strong>이라 한다.

<br/>

- 어쩔 수 없는 경우가 아나라면 <strong>사용을 자제</strong>하는 것이 좋다.
