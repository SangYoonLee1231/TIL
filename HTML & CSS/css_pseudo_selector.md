## 가상 선택자 (Pseudo Selector)

* 선택자를 작성하는 3가지 기본 방법

    * 태그의 이름을 쓰는 방법

    * 점(.)을 쓰고 클래스 이름을 쓰는 방법

    * 해시 기호(#)를 쓰고 id값을 쓰는 방법

* 이 방법 외에 다른 도구로 좀 더 세부적으로 Element를 선택하는 방법이 있다.
