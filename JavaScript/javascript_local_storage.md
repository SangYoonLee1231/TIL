## 로컬 스토리지 (Local Storage)

- 브라우저가 value를 저장하도록 해주는 API

- 작은 미니 DB 같다.

<br/>

### Local Storage의 함수

- setItem

  ```javascript
  localStorage.setItem("Key", "Value");
  ```

- getItem

  ```javascript
  localStorage.getItem("Key");
  ```

  ```
  "Value"
  ```

- removeItem

  ```javascript
  localStorage.removeItem("Key");
  ```
