## Type 자동 변환

- C계열 언어에서 사칙연산 계산시 type은 더 큰 범위를 따라가게 되어있다.

  - 정수와 실수가 피연산자로 만나면, 계산 결과의 type은 실수가 된다.

    ```c++
    double a = 1 + 3.5;
    cout << a; // 4.5
    ```
