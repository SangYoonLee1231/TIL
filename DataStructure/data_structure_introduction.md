## 🗨 자료구조(Data Structure)란

* 자료(Data)를 담는 구조

* 자료(Data)를 효율적으로 이용할 수 있도록 컴퓨터에 저장하는 방법

* Data 저장 공간 + 연산(읽기, 쓰기, 삽입, 삭제, 탐색)

* ex) 변수, 배열, 리스트, ...
