## 🗨 자료구조(Data Structure)란

* 자료(Data)를 담는 구조

* 자료(Data)에 효율적으로 접근하고 조작하기 위한 데이터의 조직, 관리, 저장 구조

* 원시 자료형(정수, 실수, 문자열 등등..)을 기반으로 만든 복합 자료형(Composite Data Type)

<br/>

* Data 저장 공간 + 연산(읽기, 쓰기, 삽입, 삭제, 탐색)

    * 자료구조의 연산은 최소 2가지가 필요하다. (삽입과 삭제)

* ex) 변수, 배열, 리스트, ...
