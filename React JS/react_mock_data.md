## 상수 데이터란?

- 변하지 않는 데이터

- React에서 데이터는 UI이다.

- 시간이 흐르면서 변하는 변수 데이터로 UI를 만들 땐 백엔드 서버에서 데이터를 받아와야 한다.

- 반면 변하지 않는 상수 데이터로 UI를 만들 땐 백엔드 서버에서 데이터를 받아올 필요가 없다.
