## Props란?

- Props는 컴포넌트의 속성값이다.

- Props는 부모 컴포넌트로부터 전달받은 데이터를 지니고 있는 객체이다.

- Props를 이용하면 부모 컴포넌트에서 자식 컴포넌트에게 어떤 값이든 (문자, 숫자, 변수, 함수 등) 전달할 수 있다.
